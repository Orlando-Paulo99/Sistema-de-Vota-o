🗳️ Sistema de Votação

Um sistema de votação simples desenvolvido em Python utilizando PySimpleGUI como interface gráfica.

O projeto simula uma eleição entre duas opções:

Candidato 1 - Backend
Candidato 2 - Frontend

O objetivo é demonstrar a criação de uma aplicação desktop com interface gráfica, coleta de informações do eleitor e registro do voto.

Observação: Este projeto não utiliza banco de dados. Os dados informados pelo usuário são utilizados apenas durante a execução da aplicação.

✨ Funcionalidades
Interface gráfica desenvolvida com PySimpleGUI.
Identificação do eleitor através de:
Nome
RG
Título de Eleitor
Escolha entre dois candidatos:
Backend
Frontend
Confirmação do voto.
Simulação simples de uma votação.
🛠 Tecnologias Utilizadas
Python 3.x
PySimpleGUI



▶ Como Executar
Utilizando o Executável

Caso não possua o Python instalado, basta abrir o arquivo:

executavel/SistemaVotacao.exe

🗳 Fluxo da Aplicação
O eleitor informa:
Nome
RG
Título de Eleitor
Após a identificação, o sistema apresenta as opções de voto:
1 - Backend
2 - Frontend
O eleitor escolhe um dos candidatos.
O sistema registra e confirma o voto.
📚 Objetivo

Este projeto foi desenvolvido para fins acadêmicos e de aprendizado, com o objetivo de praticar:

Desenvolvimento Desktop com Python;
Construção de interfaces gráficas utilizando PySimpleGUI;
Organização entre Frontend e Backend;
Lógica de programação;
Manipulação de eventos em interfaces gráficas.
